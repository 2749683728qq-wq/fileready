# ADSENSE_PRIVACY — AdSense 接入准备与隐私架构

> Phase 0 初稿 | 广告代码预留和隐私合规架构

## 1. 基本原则

AdSense 不是首版上线的前置条件。开发阶段只完成代码预留和架构准备。

没有真实 AdSense 账号时：
- 不加载真实广告
- 不使用虚假发布商 ID
- 不展示模拟成真实广告的内容
- 不创建虚假 ads.txt
- 不声称网站已获得 AdSense 批准

## 2. 环境变量

```bash
NEXT_PUBLIC_ADSENSE_ENABLED=false
NEXT_PUBLIC_ADSENSE_CLIENT_ID=
NEXT_PUBLIC_GA_ENABLED=false
NEXT_PUBLIC_GA_MEASUREMENT_ID=
```

## 3. 组件架构

创建可复用组件：

| 组件 | 用途 |
|------|------|
| `AdSenseScript` | 加载 AdSense 脚本（全站仅一次） |
| `AdSlot` | 单个广告位容器 |
| `AdPlaceholder` | 广告关闭时的占位（避免布局跳动） |
| `AdConsentGate` | 用户同意前阻止广告加载 |
| `AdPagePolicy` | 根据页面类型控制广告是否展示 |

### 3.1 组件要求

- AdSense 脚本只加载一次
- 开关关闭时不发送任何请求
- 缺少发布商 ID 时网站正常构建
- 开发环境不展示真实广告
- 广告失败不影响内容和工具
- 广告容器预留合理尺寸（减少 CLS）
- 不修改 Google 官方广告脚本核心代码
- 不追踪广告点击

## 4. 广告展示策略

### 4.1 可以展示广告的页面（审核通过后）

- 原创指南页
- 完整场景页
- 文件要求说明页
- 工具页的帮助内容区域

### 4.2 禁止展示广告的页面和区域

- 文件选择/上传/裁剪区域
- 参数设置表单内部
- 处理进度/等待状态
- 文件检查结果
- 下载页面和下载按钮附近
- 错误/空白/404 页面
- Cookie/隐私设置弹窗
- 只有操作、没有实质内容的临时页面

### 4.3 允许的广告位置

- 指南正文前半部分之后
- 长指南正文中段
- 工具完整操作区域之后
- FAQ 与相关工具之间

### 4.4 禁止的广告行为

- 广告伪装成工具卡片或处理结果
- 广告使用与主按钮相同外观
- "点击广告支持我们"等诱导文案
- 紧贴"选择文件""开始处理""下一步""下载"按钮
- 自己点击广告测试

## 5. 初始广告数量建议

| 页面类型 | 广告数量 |
|----------|----------|
| 首页 | 0 或首屏以下 1 个 |
| 工具页 | 最多 1-2 个，远离操作区 |
| 场景页 | 1-2 个 |
| 指南页 | 1-3 个（视长度） |
| 结果/下载/错误 | 0 个 |

广告和推广内容不能成为页面主体。

## 6. 自动广告策略

初期优先手动广告位。如以后启用自动广告：
- 排除所有工具操作路由和临时状态
- 排除上传、处理、结果和下载区域
- 在 AdSense 后台设置页面排除
- 先小范围测试
- 不同时开启大量锚定、插页和页内广告

## 7. ads.txt

- 开发阶段创建 `public/ads.txt.example`（只写说明，不写虚假 ID）
- 获得真实发布商 ID 后生成 `public/ads.txt`
- 确保可从主域名根路径访问

## 9. 隐私架构

### 9.1 文件隐私

首版图片和 PDF 处理在浏览器本地完成：
- 不将用户文件发送到服务器
- 不上传文件到分析工具
- 不在日志中记录文件名或文件内容
- 使用完成后释放 Object URL
- 刷新/关闭页面后清理内存中的临时数据
- 不使用 LocalStorage 保存原文件

只有当技术上确实未上传时，页面才显示："文件在你的浏览器中处理，不会上传到我们的服务器。"

### 9.2 如需服务器处理（未来）

必须：
- 明确告知用户
- 获得确认
- 说明保存时间和删除机制
- 更新隐私政策
- 不得沿用"完全本地处理"的宣传

### 9.3 Cookie 和同意管理

- 用户同意前不加载非必要广告/分析代码
- 页脚保留"隐私与 Cookie 设置"入口
- 同意可撤回
- 不使用暗黑模式诱导同意
- 接入 AdSense 时使用 Google 认证的 CMP 方案
- 具体地区规则上线前人工核对

## 10. 隐私政策内容要求

至少说明：
- 网站处理哪些数据
- 文件是否上传
- 文件处理位置
- 日志信息
- Analytics / AdSense
- Cookie
- 第三方服务
- 用户选择与撤回方式
- 数据保留
- 联系方式（使用占位符 `support@[正式域名]`）
- 政策更新时间
